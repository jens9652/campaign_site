elementært

"Let software" is an app developer located in Odense, Denmark.

They have built a chemistry learning app for educational use called "Elementært" for a third party.

We were asked to port the excisting code from iOS to Android, as well as build a webapplication for browser, running on WebGL.

The application was made in Unity and is integrated with the uni-login interface.

"Let Software" is a trusted partner, and we hope we see many future projects collaborating with them.
